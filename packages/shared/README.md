# @autoweave/shared

## Shared Utilities

Common utilities, types, and helpers used across all packages.

## Installation

This package is part of the AutoWeave monorepo. Install the entire workspace:

```bash
npm install
```

## Usage

```javascript
const { shared } = require('@autoweave/shared');
```

## Development

```bash
# Run tests
npm test

# Run in development mode
npm run dev

# Build package
npm run build
```
